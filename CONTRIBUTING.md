# Contributing

Thank you for your interest in contributing to vite-plugin-vue-hsml!

## Getting Started

1. Fork and clone the repository
2. Make sure you have [Node.js](https://nodejs.org/) >= 22 and [pnpm](https://pnpm.io/) installed
3. Run `pnpm install` to install dependencies

## Useful Commands

### Build

```sh
pnpm run build
```

### Format

```sh
pnpm run format
```

### Lint

```sh
pnpm run lint
```

### Type Check

```sh
pnpm run ts-check
```

### Test

```sh
pnpm run test
```

### Code Coverage

```sh
pnpm run coverage
```

### Preflight

Runs all checks (format, lint, type check, build, tests) in sequence:

```sh
pnpm run preflight
```
